# miniconda-notebook

To start the jupyter notebook, follow the steps for your OS below.

#linux/mac

Make sure run.sh is runnable:  
chmod +x run.sh  
  
Then execute:  
run.sh  
  
Browse to http://localhost:8888/ (should open automatically) to view your notebooks.

#windows

Execute run.bat

Browse to http://localhost:8888/ (should open automatically) to view your notebooks.
